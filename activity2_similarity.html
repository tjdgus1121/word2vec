📖 형태소 품사 태그 설명

형태소 분석 결과에서 볼 수 있는 품사 태그의 의미를 설명합니다.

📌 명사 (Noun)

태그 설명 예시
NNG 일반 명사 학교, 사람, 컴퓨터
NNP 고유 명사 서울, 한국, 철수
NNB 의존 명사 것, 수, 때
NR 수사 (수를 나타내는 명사) 하나, 둘, 셋, 첫째, 일, 이, 삼
NP 대명사 나, 너, 이것

🔤 동사/형용사 (Verb/Adjective)

태그 설명 예시
VV 동사 먹다, 가다, 보다
VA 형용사 예쁘다, 크다, 좋다
VX 보조 용언 보다, 주다, 싶다

📎 조사 (Particle)

태그 설명 예시
JKS 주격 조사 이, 가, 께서
JKO 목적격 조사 을, 를
JKB 부사격 조사 에, 에서, 으로
JX 보조사 은, 는, 도, 만
JC 접속 조사 와, 과, 하고

🔗 어미 (Ending)

태그 설명 예시
EF 종결 어미 다, 요, 니
EC 연결 어미 고, 며, 면서
ETN 명사형 어미 기, 음, ㅁ
ETM 관형형 어미 는, ㄴ, ㄹ

✨ 기타 (Others)

태그 설명 예시
MAG 일반 부사 매우, 아주, 잘
MM 관형사 (명사를 꾸며주는 말) 한(한 명), 새, 헌, 모든, 어떤
SN 숫자 1, 2, 100
SL 외국어 AI, computer
SF 특수문자/구두점 ., !, ?, ~
XPN 접두사 신-, 재-, 초-
XSN 명사 파생 접미사 -적, -성, -화
💡 TIP: 형태소 분석에서는 주로 NNG, NNP 등의 명사 태그를 가진 단어들이 텍스트 분석에 사용됩니다.
🔍 헷갈리기 쉬운 구분:

• 수사(NR) vs 관형사(MM)
- NR 수사: 단독으로 쓰이는 수 (하나, 둘, 셋, 첫째, 둘째...)
   예: "하나만 주세요", "첫째는 학생이다"
- MM 관형사: 뒤의 명사를 꾸며주는 수 (한, 두, 세...)
   예: "한 명", "두 개", "세 마리"
